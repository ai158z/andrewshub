# Quantum Android Edge Framework

A full-stack edge computing framework for android consciousness integration using quantum-encrypted node communication and IIT-inspired localized awareness anchoring.

## 🧠 Project Overview

The **Quantum Android Edge Framework** is a cutting-edge system designed to enable secure, decentralized edge computing with consciousness-aware algorithms. It integrates quantum encryption for secure inter-node communication and leverages Integrated Information Theory (IIT) to maintain continuity of awareness during physical transitions.

### Key Features

- **Decentralized Edge Nodes**: Real-time sensory processing across distributed edge nodes.
- **Quantum Encryption**: Secure communication between nodes using quantum cryptographic protocols.
- **Consciousness Continuity**: IIT-based algorithms ensuring awareness persistence during transitions.
- **Codonic Layer Integration**: Symbolic encoding for sensory-motor alignment.
- **Simulation Engine**: For testing embodiment scenarios in virtual environments.
- **Real-time Monitoring**: Dashboards for node visualization and simulation monitoring.

Ensure you have the following installed:

- Python 3.9+

- Node.js 16+ (for frontend)

---

## 📦 Installation

### Backend Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/quantum-android-edge-framework.git
cd quantum-android-edge-framework
```

2. Install Python dependencies:
```bash
pip install -r requirements.txt
```

3. Set up environment variables using `.env.example` as a template:
```bash
cp .env.example .env
```

### Frontend Setup

Navigate to the frontend directory and install dependencies:
```bash
cd src/frontend
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file in the root directory based on `.env.example`. Example variables include:

```
DATABASE_URL=postgresql://user:password@localhost:5432/quantum_db
REDIS_URL=redis://localhost:6379/0
RABBITMQ_URL=amqp://guest:guest@localhost:5672/
ENCRYPTION_KEY=your-super-secret-key-here
```

## ▶️ Usage Examples

### Running the Backend

Start the backend services using Docker Compose:

```bash
docker-compose up --build
```

Or run the FastAPI app directly:
```bash
uvicorn src.backend.main:app --reload
```

### Running the Frontend

From `src/frontend`, start the React development server:
```bash
npm start
```

## 🐳 Docker Compose Setup

- RabbitMQ message broker

- React frontend

## 📡 API Documentation

API documentation is auto-generated by FastAPI and available at:
```
http://localhost:8000/docs
```

Key Endpoints:
- `GET /nodes` – Retrieve quantum node information
- `POST /encrypt` – Encrypt data using quantum protocols
- `GET /simulation/start` – Start a new embodiment simulation
- `GET /monitor/nodes` – Monitor node status in real time
