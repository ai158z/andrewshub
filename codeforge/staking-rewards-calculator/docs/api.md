**Parameters:**
- `apy` (float): Annual Percentage Yield as a decimal (e.g., 0.05 for 5%)
- `compound_frequency` (int, optional): Number of compounding periods per year. Defaults to 1.
- `penalty_rate` (float, optional): Penalty rate as decimal for early withdrawal. Defaults to 0.0.

#### calculate_rewards()
